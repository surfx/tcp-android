
# About

Server TCP para controle windows - mouse, click, som (sound), shutdown

## Create project

```
dotnet new console --name tcpserver_csharp
```

## Dependência externa

```
dotnet add package AudioSwitcher.AudioApi.CoreAudio --version 3.0.0.1
```

## Start

```
dotnet run
```
